# Machine Learning: Kompletny Bootcamp – część 1

Repozytorium zawiera materiały, notatki oraz przykładowe projekty przygotowane podczas realizacji kursu **"Machine Learning: Kompletny Bootcamp - od podstawowych koncepcji do zaawansowanych algorytmów! (część 1)"** dostępnego na platformie Udemy.

## Zakres kursu

W pierwszej części kursu omawiane są podstawowe zagadnienia związane z uczeniem maszynowym, w tym:

- wprowadzenie do Machine Learning
- przygotowanie środowiska pracy
- podstawy pracy z Pythonem w kontekście ML
- analiza i przygotowanie danych
- wizualizacja danych
- podstawowe algorytmy uczenia maszynowego
- trenowanie i ewaluacja modeli

## Technologie

W projekcie wykorzystywane są m.in.:

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Cel repozytorium

Celem repozytorium jest:
- utrwalenie wiedzy zdobytej podczas kursu
- przechowywanie ćwiczeń i przykładów
- dokumentowanie postępów w nauce Machine Learning
